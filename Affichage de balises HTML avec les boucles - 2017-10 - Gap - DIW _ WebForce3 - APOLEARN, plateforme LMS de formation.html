L'instruction break permet de sortir d'une structure for, foreach, while, do-while.

Soit la boucle suivante

for($i=0; $i<10; $i++) {
    echo $i.'<br/>';

Fiche de révision

Répéter des instructions avec les boucles

Dans la réalisation de votre site dynamique, vous serez amené à afficher les mêmes structures de balises pour afficher un menu, une liste d'options ou encore une liste d'articles.
Il sera trop long et fastidieux d'écrire toutes ces balises une par une! Pour cela, nous allons utiliser les différentes instructions de boucle proposées par PHP.

Que se passe-t-il dans une boucle ?

Les instructions sont exécutées dans l'ordre, de haut en bas.
A la fin des instructions, on retourne à la première instruction et ainsi de suite jusqu'à ce que l'on sorte de la boucle.
Le plus gros risque dans une boucle, c'est de ne jamais en sortir en ayant mal défini les conditions de sortie (cas de boucle infinie).
Cela peut se produire avec les instructions de boucle while ou for mais pas avec foreach.

Boucle while

Afficher 10 boutons avec la boucle while

$nbbuttons = 0;
while ($nbbuttons < 10) {
    echo '<button>Mon button</button>';
    $nbbuttons++;
}

while peut se traduire par « tant que ». Ici, on demande à PHP : tant que $nbbuttons est inférieur à 10, on affiche un bouton.
Au 10e passage dans la boucle, nous sortons de celle-ci car $nbbuttons sera supérieur à 10. C'est notre condition de sortie.

Boucle for

Même exercice avec la boucle for

for ($i = 0, $i < 10, $i++) {
    echo '<button>Mon button</button>';
}

for peut se traduire par « pour ». Ici, on demande à PHP : pour i égal à 0 jusqu'à i inférieur à 10, exécuter les instructions de la boucle tout en incrémentant i de 1 à chaque itération.
La boucle for s'occupe d'incrémenter la valeur de i sans que nous ayons à le faire dans les instructions.

Boucle foreach

La boucle foreach est la plus utilisée en PHP pour l'affichage des balises, car elle correspond plus aux besoins d'un développeur. En effet, il n'y a pas trop de sens à afficher 10 boutons sans plus d'information. La boucle foreach est idéal car elle parcours un tableau, et dans notre cas, un tableau qui contient toutes les informations pour l'affichage des balises.

Exemple : Afficher les liens d'action d'un élément.

// Tableau qui contient les informations des liens à afficher
$actions = array(
    '0' => array('action' => 'Créer', 'lien' => '/create/'),
    '1' => array('action' => 'Surpprimer', 'lien' => '/delete/'),
);

// Parcours du tableau pour afficher les liens dynamiques
foreach($actions as $key => $value) {
     echo '<a href="'.$value['lien'].'">'.$value['action'].'</a>';
     echo '<br/>';
}
// Affiche <a href="/create/">Créer</a><br/><a href="/delete/">Supprimer</a><br/>

Instruction break

L'instruction break permet de sortir d'une structure for, foreach, while, do-while.
Nous pouvons l'utiliser pour une recherche par exemple, une fois l'item trouvé, cela ne sert plus à rien de continuer à chercher.
