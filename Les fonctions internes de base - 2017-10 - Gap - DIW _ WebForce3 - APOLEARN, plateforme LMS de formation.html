Questions

À quoi sert la fonction count ?

Choisissez la meilleure réponse.

Bonne réponse

Explications

La fonction count(…) reçoit en paramètre une variable de type array et retourne le nombre de valeurs contenues dans ce tableau.


Exemple


$a = array(2,4,6);

echo count($a); // affichera 3 car il y a 3 valeurs dans ce tableau

À quoi sert la fonction time ?

Choisissez la meilleure réponse.

Bonne réponse

Explications

La fonction time permet de renvoyer le nombre de secondes écoulées depuis le 1er Janvier 1970, appelé timestamp.
La fonction date permet quand à elle de retourner une date formatée de manière spécifique, que ce soit la date au moment de l'exécution de l'instruction ou bien celle d'un timestamp spécifié en paramètre.


Exemple


$T = time();

echo $T; // Affichera quelque chose comme 1445394540

echo date('d / m / Y — H:i', $T); // Affichera: 21 / 10 / 2015 — 4:29, soit la date représentée par le timestamp contenu dans $T

La fonction empty(…) renvoie true si la variable spécifiée est…

Choisissez la meilleure réponse.

Bonne réponse

Explications

La fonction empty(…) permet de vérifier que la variable spécifiée est soit:



  • égale à 0, 0.0 ou "0"

  • égale à false

  • égale à une chaîne de caractère vide ""

  • égale à un array vide

  • non définie

  • fait référence à une autre variable remplissant l'un de ces critères

Quelle fonction permet de vérifier qu'une variable est bien définie (même si elle est vide) ?

Choisissez la meilleure réponse.

Bonne réponse

Explications

La fonction isset(…) renvoie true si la ou les variable spécifiées sont définies, même si elles sont vides.


Exemple


$a = 'Hello';



if (isset($a)) {

    echo '$a est défini';

}



if (isset($a, $b)) {

    echo '$a et $b sont définis'; // Ici, cette phrase ne s'affichera pas car $b n'existe pas, donc isset() renvoie false

}
